# IoT-Telemetry
Приложение для сбора данных с промышленных датчиков по протоколу Modbus (TCP) и их визуализации в реальном времени.

- App - графическое приложение для отображения и взаимодействия с датчиками;
- Modbus-client - программа для ESP-32 контроллера на котором расположены датчики;
- Modbus-server - программа для серверного контроллера для приёма/отправки команд от графического приложения.
